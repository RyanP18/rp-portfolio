# Ryan Popp's Portfolio

## Developing

1. To serve the website locally, from a terminal, run:

```javascript
npx serve
```

2. Open a browser and navigate to [http://localhost:3000.com](http://localhost:3000.com)
3. Edit the code
4. Save your changes
5. Refresh your browser
